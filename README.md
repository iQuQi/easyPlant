# 😀 프로젝트 소개

### 식물에게 물 주는 날들을 보기 쉽게 정리해 드립니다. 🗒

- 물을 주고 나면 물뿌리개를 눌러서 식물 행복도를 올려 보세요! 행복도는 물을 빠지지 않고 줄수록 높이 올라갑니다.
- 식물들의 물 주는 날을 알람을 받을 수 있어요!

### 식물과 함께 성장하기 📈

- 키우고 있는 식물과 함께 레벨을 올려봅시다.
- 식물을 오래 행복하게 키워서 경험치를 쌓을 수 있어요!

### 식물 정보 검색 🔎

- '농사로' 의 공공데이터를 사용하고 있어 취향별 검색으로 나에게 맞는 정보를 찾을 수 있습니다.
- 다양한 식물의 정보를 쉽고 빠르게 획득해보세요!

### 반려식물 갤러리 📓

- 반려식물들을 보기 쉽게 목록을 보여줘요.
- 식물들을 위한 다이어리를 작성해보세요!
- 식물의 성장과정을 모아 볼 수 있습니다!
